# LLM4SSC — Agent Guide

Academic paper: "LLMs for Software Supply Chain Security: Boon or Bane?"
AI Ethics and Governance course, Fudan University. NeurIPS 2025 template.

## Build

```sh
make              # produces main.pdf (runs pdflatex×2 + bibtex + pdflatex)
```

Requires `pdflatex`, `bibtex`, and the `neurips_2025` style package (not in repo; download from NeurIPS).

## Structure

| Path | Purpose |
|---|---|
| `main.tex` | Paper entrypoint, preamble, title, abstract, bibliography |
| `src/*.tex` | Per-section content (add new sections here) |
| `dataset/*.bib` | Bibliographic entries by source (acm, arxiv, ieee, sciencedirect, misc) |
| `pdfs/*/` | Reference PDFs mirroring the .bib sources |

## Conventions

- Add `.tex` files under `src/` and include with `\input{src/...}` in `main.tex`
- Add `.bib` entries to the appropriate file in `dataset/` — all are concatenated into `main.bib` at build time
- Commit reference PDFs to `pdfs/<source>/` alongside the `.bib` entries
- `.gitignore` excludes `pdfs/`, `main.bib`, and all LaTeX build artifacts (`*.aux`, `*.log`, `*.pdf`, etc.)
- `main.bib` is generated by `make` (concatenation of `dataset/*.bib`) — **do not edit it directly**

## Common tasks

- **Add citation**: add `\cite{key}` in `.tex`, ensure `key` exists in a `dataset/*.bib` file
- **Add section**: create `src/<name>.tex`, add `\input{src/<name>}` in `main.tex`
- **Clean build**: `rm -f main.bib *.aux *.log *.out *.bbl *.blg main.pdf && make`
